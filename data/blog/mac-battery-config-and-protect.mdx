## 前言：为什么电池管理如此重要？

现代 MacBook 作为移动办公的核心工具，电池健康直接影响使用体验和设备寿命。锂电池的化学特性决定了其在高电量状态下会加速老化，而合理的充电管理可以显著延长电池使用寿命。

苹果官方数据显示，MacBook 电池在正常使用条件下，经过 1000 次充电循环后仍能保持原始容量的 80%。通过科学的充电管理，我们可以进一步优化这一表现。

## 系统内置：macOS 优化电池充电功能

### 工作原理

苹果在 macOS 中内置了**"优化电池充电"**功能，该功能通过机器学习算法分析你的使用习惯：

- **智能学习**：系统会记录你的充电时间和使用模式
- **预测性充电**：在预计你需要使用设备前完成充电
- **80% 暂停**：在某些情况下将充电暂停在 80%，直到你需要使用

### 启用方式

1. 打开 **系统设置** → **电池** → **电池健康**
2. 点击电池健康旁边的 **(i)** 图标
3. 确保 **"优化电池充电"** 已开启

> [!NOTE] > **苹果官方说明**：[关于 MacBook 的优化电池充电](https://support.apple.com/zh-cn/102589)
>
> 此功能需要一定时间学习你的使用习惯，初期可能不会立即生效。

## 第三方工具：精确控制充电上限

> [!NOTE] > **开始之前，需要在终端先安装 HomeBrew**
>
> ```
> /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
> ```

**当系统内置功能无法满足需求时，第三方工具提供了更精确的控制选项。**

### 1. BCLM - 命令行专业工具

**BCLM**（Battery Charge Level Max）是一个轻量级命令行工具，直接操作系统管理控制器（SMC）来控制充电上限。

#### 安装方法

```bash
# 使用 Homebrew 安装（推荐）
brew tap zackelia/formulae
brew install bclm
```

#### 使用指南

**Intel 芯片 MacBook：**

```bash
# 设置充电上限为 77%（实际显示约 80%）
sudo bclm write 77

# 查看当前设置
bclm read

# 查看当前电池状态
bclm status
```

**Apple 芯片 MacBook：**

```bash
# 仅支持 80 或 100 两个值
sudo bclm write 80

# 恢复到 100%
sudo bclm write 100
```

> [!IMPORTANT] > **兼容性说明**：
>
> - Intel 芯片：支持 1-100 任意值
> - Apple 芯片：仅支持 80/100，需要 macOS 13.0-14.x
> - macOS 15+ 的 Apple 芯片暂不支持

#### 持久化设置

为防止系统重启后设置丢失：

```bash
# 创建持久化服务
sudo bclm persist

# 检查服务状态
sudo launchctl list | grep bclm
```

**项目地址**：[GitHub - zackelia/bclm](https://github.com/zackelia/bclm)

### 2. AlDente - 图形界面的优雅选择

**AlDente** 提供直观的图形界面，是最受欢迎的 Mac 电池管理工具之一。

#### 主要功能

- **精确控制**：支持 20%-100% 任意充电上限
- **主动放电**：即使插电也可使用电池供电
- **详细统计**：电池健康度、循环次数等信息
- **美观界面**：菜单栏集成，操作简便

#### 版本对比

| 功能       | 免费版 | Pro 版 |
| ---------- | ------ | ------ |
| 基础限电   | ✅     | ✅     |
| 放电功能   | ❌     | ✅     |
| 详细统计   | ❌     | ✅     |
| 自定义图标 | ❌     | ✅     |

#### 使用技巧

1. **日常办公**：设置 80% 上限
2. **长期存储**：设置 50% 并启用放电功能
3. **出差前**：临时调整至 100%

**官网地址**：[AlDente - Charge Limiter](https://apphousekitchen.com/)

### 3. Battery Toolkit - Apple 芯片专用

专为 Apple 芯片 MacBook 设计的现代化电池管理工具。

#### 安装步骤

```bash
# Homebrew 安装
brew tap mhaeuser/mhaeuser
brew install battery-toolkit
```

#### 首次运行设置

由于未经 Apple 公证，需要手动允许运行：

**macOS 14 及以下：**

1. 右键点击应用 → **打开**
2. 在弹窗中选择 **"打开"**

**macOS 15 及以上：**

1. 尝试打开应用（会被阻止）
2. 前往 **系统设置** → **隐私与安全性**
3. 滚动到底部，点击 **"仍然打开"**
4. 完成身份验证

#### ⚠️ 重要配置

为确保不会发生干扰，使用前最好关闭系统自带的优化充电功能：

**系统设置** → **电池** → **电池健康** → **(i)** → **优化电池充电** → **关闭**

**项目地址**：[GitHub - mhaeuser/Battery-Toolkit](https://github.com/mhaeuser/Battery-Toolkit)

## 充电接口对比：MagSafe vs USB-C

### MagSafe 3 充电器

**优势：**

- **快速充电**：支持最高功率充电
- **磁吸设计**：防止意外拉扯损坏
- **专用优化**：苹果原生充电协议

**适用场景：**

- 长时间桌面使用
- 需要快速充电时
- 多人共用环境

### Type-C 充电

**优势：**

- **通用性强**：兼容多种设备
- **便携性好**：一线多用
- **成本较低**：第三方选择多

**注意事项：**

- 确保使用支持 PD（Power Delivery）协议的充电器
- 功率不足可能导致充电缓慢
- 部分廉价充电器可能影响电池健康

### MagSafe 和 Type-C 同时连接

> [!NOTE] > **根据苹果官方说明**：[为 MacBook Air 或 MacBook Pro 充电](https://support.apple.com/zh-cn/102397)

**Mac 一次只能通过一个端口进行充电，因此将多个电源适配器连接到 USB-C 端口和 MagSafe 3 端口（如果有）不会加快 Mac 的充电速度。不同 USB-C 充电线支持的最大功率 (W) 不同，因此请确保选择正确的充电线。如果你连接了多个电源，则 Mac 只会通过供电能力最高的充电线和电源适配器或显示器进行充电。**

**供电优先策略示例：**

| 连接方式         | MagSafe 功率 | Type-C 功率 | 实际使用电源           |
| ---------------- | ------------ | ----------- | ---------------------- |
| MagSafe + Type-C | 140W         | 100W        | ✅ MagSafe（140W）     |
| MagSafe + Type-C | 67W          | 100W        | ✅ Type-C（100W）      |
| MagSafe + Type-C | 96W          | 96W         | ✅ MagSafe（默认优先） |

### 如何查看 MagSafe 3 和 Type-C 的供电功率

**方法一：使用 macOS 自带系统报告查看（靠自己）**

#### 具体步骤

- 1. 点击左上角苹果图标  → 关于本机
- 2. 选择 **更多信息 → 通用 → 系统报告**
- 3. 在左侧找到 **电源（Power）**
- 4. 查看 **交流充电器信息** 一栏，即可得知当前使用的电源类型与功率
- 5. 交替插拔 MagSafe 与 Type-C 电源，观察功率变化并进行验证
     > 示例截图如下：
     > ![battery](https://image.jxzsite.com/i/2025/05/27/sqzbnq.png)

**方法二：不耻下问（还是靠自己）**

#### 具体步骤

- 1. 向 **Apple 官方客服** 咨询，提供你的设备型号，他们能帮你确认对应电源规格（如 MagSafe 支持最高 140W）
- 2. 向 **Type-C 电源适配器或线材卖家** 询问其功率规格，尤其关注是否支持 PD 协议及最大电流（如 20V⨉5A = 100W）
